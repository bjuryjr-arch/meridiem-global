Refine the design system to use a consistent Meridiem brand color palette and improve visual hierarchy across the application.

---

## 🎨 BRAND COLOR SYSTEM

Use the following colors with clear roles:

Primary background:

* #f8f3e4 (soft cream) for all main page backgrounds

Primary brand color:

* #091929 (deep navy) for:

  * headings
  * primary text
  * navigation
  * primary UI emphasis

Secondary brand color:

* #394452 (muted slate) for:

  * secondary text
  * icons
  * supporting UI elements
  * labels and less prominent content

---

## 🧱 LAYOUT + CARD DESIGN

* Cards should be white or slightly lighter than the background to create separation
* Ensure cards clearly stand out from the cream background
* Add subtle shadows or soft borders to separate sections
* Maintain a clean, minimal SaaS layout

---

## 📊 VISUAL HIERARCHY + CONTRAST IMPROVEMENTS

Improve visual contrast and hierarchy on the cream (#f8f3e4) background:

* Ensure cards stand out clearly from the background
* Add subtle shadows or borders to separate sections
* Improve spacing between sections
* Increase contrast between text and background for readability
* Make key data points (metrics, totals, statuses) visually prominent
* Use navy (#091929) for emphasis and important information
* Use slate (#394452) for supporting or secondary information

---

## 🔘 BUTTONS + INTERACTIONS

* Primary buttons:

  * Use #091929 background with light text
* Secondary buttons:

  * Outline style using #091929 or #394452
* Ensure buttons are visually distinct and easy to tap (mobile-first)

---

## 📱 MOBILE UX

* Maintain clean spacing and padding
* Ensure layouts feel breathable and not cramped
* Keep tap targets large and clear
* Ensure color contrast remains strong on smaller screens

---

## ⚠️ IMPORTANT RULES

* Do NOT introduce new random colors outside this palette
* Do NOT change layout structure
* Focus only on improving:

  * color consistency
  * contrast
  * spacing
  * visual clarity

---

## 🎯 GOAL

The UI should feel:

* premium
* clean
* modern SaaS
* consistent with Meridiem branding

Not generic, not flat, and not overly bright.
