Refine the header CTA buttons to improve visual hierarchy and create a more premium, modern SaaS appearance.

---

## GOAL

Create a clear distinction between:

* Primary CTA: "See if you're ready to scale"
* Secondary CTA: "Client Login"

The primary CTA must visually dominate.

---

## PRIMARY CTA (See if you're ready to scale)

* Use a filled button with background color #091929
* Use white or very light text
* Increase padding slightly for a more substantial feel
* Add subtle shadow or elevation for depth
* Keep border radius moderate (not overly rounded)
* Ensure it stands out clearly from the header background

---

## SECONDARY CTA (Client Login)

* Reduce visual weight significantly

* Use one of the following styles:
  OPTION A: simple text button (no border)
  OPTION B: thin outline button with very subtle border

* Use #394452 or slightly muted navy for text

* Reduce padding compared to primary CTA

* Make it feel like a utility action, not a conversion action

---

## SPACING + ALIGNMENT

* Add clear spacing between the two buttons
* Ensure they are aligned vertically and horizontally
* Avoid overly large pill shapes

---

## DESIGN CLEANUP

* Reduce border thickness on buttons
* Avoid overly rounded "bubble" appearance
* Ensure the buttons feel sharp, clean, and intentional
* Match overall Meridiem brand (premium, minimal, controlled)

---

## MOBILE

* Keep primary CTA visible
* Move Client Login to menu if needed
* Maintain clean spacing and tap targets

---

## FINAL FEEL

The header should feel:

* intentional
* high-end SaaS
* not generic or template-like

Do not change button labels or layout — only refine styling, hierarchy, and spacing.
